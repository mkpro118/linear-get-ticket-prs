# Get Tickets

```
Returns ticket identifiers matching specified filters, one per line

Usage: get-tickets [OPTIONS]

Options:
  -l, --label <LABELS>        
  -s, --status <STATUSES>     
  -a, --assignee <ASSIGNEES>  
  -n, --limit <LIMIT>         
  -k, --api-key <API_KEY>     
  -h, --help                  Print help

```
